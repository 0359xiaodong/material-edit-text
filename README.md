Usage
-----



Download
--------

Download the [latest JAR]() or grab via Maven:
```xml
<dependency>
  <groupId>com.pombo</groupId>
  <artifactId>materialedittext</artifactId>
  <version>1.0.0</version>
</dependency>
```

or Gradle:
```groovy
compile 'com.pombo:materialedittext:1.0.0'
```
